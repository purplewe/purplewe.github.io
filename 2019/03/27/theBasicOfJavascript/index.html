解构运算符、拓展运算符、rest运算符

解构运算符：作用是快速获取对象或数组中的元素。

深拷贝
var a = [1, 2];
var b = a;
var c = […a];
console.log(a === b);//true
console.log(a === c);//false

字符串转数组
var str = ‘hello’;
var exstr = […str];
console.log(exstr);//[‘h’, ‘e’, ‘l’, ‘l’, ‘o’]
`

rest运算符：作用与拓展运算符相反
